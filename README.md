# Campus Connect Hub

A comprehensive platform designed to streamline campus life and academic management.

## Project Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository**
   ```sh
   git clone https://github.com/BendalamAkshaya/campus-connect-hub.git
   ```

2. **Navigate to the project directory**
   ```sh
   cd campus-connect-hub
   ```

3. **Install dependencies**
   ```sh
   # If you use npm
   npm install
   ```

4. **Start the development server**
   ```sh
   npm run dev
   ```

## Key Features

- **Attendance Tracker**: Keep track of your academic presence.
- **Assignment System**: Manage and submit assignments efficiently.
- **Study Material Hub**: Access and share study resources.
- **AI Study Assistant**: Get help from intelligent study tools.
- **Smart Timetable**: Stay organized with your classes.
- **Internship Hub**: Discover and apply to campus opportunities.
- **Community Forum**: Connect with peers and share knowledge.
